Week 3 of 4-week Pre-competition Mesocycle

Competition Program

Coaches' Notes
  • Higher volume
  • Tuesday and Sunday are long workouts, so people should warm up before the sessions.
Athlete Notes
  • Tuesday's workout is long and tiring.
  • Sunday's workout includes some heavy lifts.

Tuesday

  1. Drop Snatch 5x3

  2. Power Snatch 60x3 65x3 70x3 75x3 80x3

  3. PC+3PJ 60x3 65x3 70x3 75x3 80x3

  4. Back Squat 60(2x6) 65(2x5) 70x4 75x3 80(2x3)

  5. Snatch Pull 85(2x3) 90(2x2)

  6. Weighted Planks 3x1 minute

Thursday

  1. Snatch 60x3 65x3 70x3 75(2x3) 80x3

  2. C&J 55x3 60x3 65x3 70(2x3) 75x3

  3. Front Squat 60x4 65x4 70x4 75x3 80x3 85x3

  4. Snatch Pull 85(2x3) 90(2x2)

  5. 4 rounds:

    • Abalakov Jump x10
    • Barbell Rows x10
    • Wrist curls x15-18

Sunday

  1. Snatch 65x3 70(2x3) 75x3 80x3 85x3 90x2 80x3 90x2

  2. C&J 55x3 60x3 65x3 70x3 75x3 80x3 85x3 80x2 90x2

  3. Back Squat 60x4 65x4 70x3 75x3 80(2x3) 85x2 90x2

  4. Snatch Pull 85(2x3) 90(2x2)

  5. Pullups 4x10

  6. Weighted Planks 3x1 minute

Day 4

Monday, Wednesday, Friday, or Saturday

  1. Snatch Grip Power Jerk behind neck 65x3 70x3 75x3 80x3 85x3 90x3 (based on Snatch)

  2. Muscle clean + Push Press 6x3

  3. Back Squat 70x3 75x3 80x3

  4. 4 rounds:

    • Clean Grip BN Press x10
    • Single Leg Abalakov Jump x10
    • Side Plank banded clamshell x10
